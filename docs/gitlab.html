Connect Pangolin.js Visual Review with a GitLab repository:

<script
  data-backend-type="gitlab"
  data-backend-url="https://gitlab.com/api/v4"
  data-repository="mvsde/visual-review-demo-issues"
  id="pangolinjs-visual-review"
  src="http://cdn.jsdelivr.net/npm/@pangolinjs/visual-review"
  defer
></script>

The issues will be created in this demo repository.

To create a new issue, a personal access token is required.
